# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Tier Maker - a full-service tier list creation and sharing web application (similar to tiermaker.com).

## Tech Stack

- **Framework**: Next.js (App Router) with TypeScript
- **Styling**: Tailwind CSS
- **Backend/Auth/Storage**: Supabase (DB, Auth, Storage)
- **Drag & Drop**: dnd-kit
- **Deployment**: Vercel

## External Services

- Supabase MCP server is configured in `.claude/settings.local.json`
